# toy-robot

# Running the app

```
$ npm install
$ npm start
```

# Running tests
```
npm test
```

# REPL commands

```
PLACE --x {value} --y {value} --f {facing_value}

$ PLACE --x 2 --y 1 --f NORTH
$ MOVE
$ LEFT
$ RIGHT
$ REPORT
```
